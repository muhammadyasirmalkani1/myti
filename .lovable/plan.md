

# AI Wallet Landing Page

## Overview
A clean, minimal landing page for a crypto/Web3 AI wallet product, featuring the uploaded video as a background element in the hero section.

## Sections

### 1. Navigation Bar
- Logo/brand name on the left
- Navigation links (Features, How It Works, etc.)
- "Get Started" / "Launch App" CTA button on the right
- Clean, minimal sticky header with subtle backdrop blur

### 2. Hero Section (with Video Background)
- The uploaded **Setup-Background.mp4** plays as a looping, muted background video behind the hero content
- Bold headline (e.g., "The AI-Powered Wallet for Web3")
- Short subheadline describing the value proposition
- Primary CTA button ("Get Started" / "Connect Wallet")
- Subtle overlay on the video to ensure text readability

### 3. Features Section
- 3-4 feature cards highlighting key AI wallet capabilities:
  - AI-powered transaction insights
  - Smart portfolio management
  - Secure multi-chain support
  - Automated DeFi strategies
- Clean card layout with icons and brief descriptions

### 4. Call-to-Action Section
- Final CTA block encouraging users to get started
- Email signup or "Launch App" button
- Clean background with subtle styling

### 5. Footer
- Minimal footer with links (Privacy, Terms, Social icons)

## Design Details
- **Light/white base** with subtle gradients and clean typography
- Smooth scroll animations as sections come into view
- Mobile-responsive layout
- The video background will be muted, looping, and auto-playing with a light overlay
